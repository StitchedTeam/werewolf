ファイルの説明
ptCollage.exe  : ピストンコラージュ本体（作曲ツール）
ptVoice.exe    : ピストンボイス本体（専用音源作成ツール）
ptNoise.exe    : ピストンノイズ本体（専用音源作成ツール）
ptPlayer.exe   : ピストンプレイヤー（曲再生専用ツール・再配布可）
pxtone.dll     : ピストンプレイヤーの起動に必要なDLL。(再配布可)
pxtoneTool.dll : 各編集ソフトの起動に必要なDLL。(プレイヤーには必要なし)
manual_ptc.html: ピストンコラージュマニュアル（お読みください）
manual_ptv.html: ピストンボイスマニュアル
manual_ptn.txt : ピストンノイズ簡易マニュアル
japanese.ico   : このファイルを削除すると怪しい英語モードになります。

以下はフォルダ
my_material    : 試作用の簡易音源が入ってます
my_project     : 作った曲を保存するフォルダです。(空です)
pxtone_include : ピストンDLLを組み込むためのlibとヘッダファイルが入ってます。
sample         : サンプル曲です。

アンインストールは、このテキストが入ってるフォルダを
まるごと削除するだけでOKです。

このアプリケーションはフリーソフトです。
実行中の事故は使用者各自の責任とします。
（制作に悪意が無いことをご理解ください）

今後もバージョンアップしていく予定です。
バグ報告などいただけると助かります。
